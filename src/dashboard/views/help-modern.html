üõ°Ô∏è Core Security Features

Advanced protection systems to keep your Discord server safe

Anti-Raid Protection

Automatically detects and prevents coordinated raid attacks by monitoring suspicious join patterns.

  • Join Rate Monitoring: Tracks members joining per minute
  • Auto-Lockdown: Automatically locks server when threshold exceeded (default: 10 joins/min)
  • Suspicious Pattern Detection: Identifies bot raids and coordinated attacks
  • Real-Time Alerts: Notifies staff immediately via configured channel
  • Account Age Filtering: Optionally kicks accounts under X days old
/antiraid enable /antiraid threshold [number] /antiraid action [kick/ban/timeout]

Anti-Spam Protection

Prevents message spam, duplicate content, and mass mentions with intelligent detection.

  • Duplicate Message Detection: Blocks repeated messages (max 5 in 10 seconds)
  • Mention Spam Prevention: Limits mass mentions (max 5 per message)
  • Rapid Fire Detection: Identifies unusually fast message sending
  • Auto-Moderation: Automatically deletes spam and warns users
  • Link Spam Filter: Detects and blocks repetitive link posting
/antispam enable /antispam configure /antispam whitelist @role

Anti-Nuke Protection

Prevents compromised accounts from destroying your server with advanced action limits.

  • Channel Protection: Limits channel deletions (max 5/min)
  • Role Protection: Prevents mass role deletion (max 5/min)
  • Ban Limits: Stops mass ban attacks (max 10/min)
  • Permission Monitoring: Tracks bulk permission changes
  • Whitelist System: Bypass limits for trusted administrators
/antinuke enable /antinuke whitelist @user /antinuke settings

Anti-Phishing Protection

Scans messages for malicious links and phishing attempts using multiple databases.

  • Link Scanning: Checks all URLs against known phishing databases
  • Pattern Analysis: Detects suspicious link structures
  • Auto-Delete: Instantly removes dangerous links
  • User Warnings: Alerts users about blocked attempts
  • API Integration: Optional VirusTotal, URLVoid, Google Safe Browsing
/antiphishing enable /antiphishing check [url] /antiphishing whitelist [domain]

Lockdown System

Emergency server lockdown with multiple modes to stop active threats immediately.

  • Full Lockdown: Prevents all new messages server-wide
  • Soft Lockdown: Restricts to verified members only
  • Channel Lockdown: Lock specific channels individually
  • Auto-Unlock: Schedule automatic unlocking after X minutes
  • Raid Response: Integrates with anti-raid for automatic lockdown
/lockdown [mode] [duration] /lockdown channel #channel /unlockdown

Verification System

Multi-layer verification to prevent bot accounts and ensure legitimate members.

  • Captcha Verification: Solve captcha to gain access
  • Button Verification: Simple click-to-verify system
  • Manual Approval: Staff-reviewed verification for sensitive servers
  • Role Assignment: Automatically grants verified role on success
  • DM Verification: Send verification prompts via direct message
/verification setup /verification mode [captcha/button/manual] /verification role @role

üîê Privacy Policy

How we collect, use, and protect your data

Data Collection

DarkLock collects and stores the following data to provide security and moderation services:

  • Server Data: Server ID, server name, member count, channel information
  • User Data: User ID, username, discriminator, avatar URL, roles
  • Message Data: Message content for anti-spam/phishing detection (not permanently stored)
  • Moderation Logs: Ban/kick/warn records, moderator actions, case notes
  • Security Incidents: Join patterns, suspicious activities, threat data
  • Settings: Bot configuration, feature toggles, custom settings per server

Data Usage

We use collected data exclusively for the following purposes:

  • Security Protection: Detecting raids, spam, phishing, and malicious activity
  • Moderation: Tracking warnings, bans, and moderator actions
  • Analytics: Server statistics, usage metrics, and activity trends (dashboard only)
  • Bot Functionality: Welcome messages, verification, leveling, economy features
  • Support: Ticket system, AI assistance (if enabled)

Data Protection

We implement strong security measures to protect your data:

  • Encryption: Dashboard authentication uses JWT tokens with HTTPS
  • Access Control: Role-based permissions ensure only authorized users access data
  • Local Storage: Data stored in SQLite database on your server (self-hosted)
  • No Third-Party Sharing: Your data is never sold or shared with third parties
  • Automatic Cleanup: Old logs and unused data automatically deleted after 90 days

Data Deletion

You have full control over your data:

  • Remove Bot: Kicking the bot deletes all server data within 48 hours
  • Manual Request: Contact bot administrator to request immediate data deletion
  • User Data: Individual users can request deletion of their personal data
  • Export Data: Server admins can export all data via dashboard backup feature

Third-Party Services

DarkLock may integrate with these optional services:

  • VirusTotal: Link scanning for phishing detection - URLs checked against VirusTotal database
  • Google Safe Browsing: Additional phishing protection - URLs verified with Google
  • Stripe: Payment processing for premium features - handles payment data securely (never stored by bot)

Note: All third-party integrations are optional and can be disabled. Review each service's privacy policy before enabling.

Legal Compliance

  • GDPR Compliant: Users in EU have right to access, rectify, and delete their data
  • COPPA Compliant: No collection of data from users under 13
  • Discord ToS: Fully compliant with Discord's Terms of Service and Developer Terms
  • Data Retention: Moderation logs kept for 90 days, security logs for 30 days

Contact & Questions

For privacy-related questions or data requests:

  • Email: Contact server administrator or bot developer
  • Discord Support: Join bot support server for assistance
  • Data Requests: Response within 30 days per GDPR requirements
