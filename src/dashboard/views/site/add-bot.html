Add DarkLock to Your Server

Start protecting your community in just a few clicks

Ready to Secure Your Server?

Click the button below to add DarkLock to your Discord server. You'll be redirected to Discord to authorize the bot.

Add to Discord

How It Works

1

Click Add to Discord

Click the button above to start the authorization process.

2

Select Your Server

Choose the server you want to add DarkLock to from the dropdown.

3

Authorize Permissions

Review and authorize the required permissions for full functionality.

4

Run /setup

Use the /setup command in your server to configure DarkLock.

5

You're Protected!

DarkLock is now actively protecting your server 24/7.

Required Permissions

DarkLock requires Administrator permission to function properly. This allows the bot to:

Manage channels
Manage roles
Ban/kick members
View audit logs
Manage messages
Create invites

DarkLock only uses permissions when necessary for security features.
