Introduction

DarkLock is a comprehensive security and moderation bot for Discord servers. It provides real-time protection against raids, spam, phishing, and other threats while offering powerful moderation and utility features.

This documentation will help you set up and configure DarkLock to best protect your community.

Quick Start

Get DarkLock up and running in minutes:

  1. Add DarkLock to your server using the invite link
  2. Run /setup in your server to start the setup wizard
  3. Follow the wizard to configure basic settings
  4. Use the dashboard for advanced configuration

Initial Setup

After adding DarkLock to your server, run the /setup command to access the interactive setup wizard.

Setup Wizard Options

  • Log Channel: Select where security events are logged
  • Mod Roles: Define which roles can use moderation commands
  • Mute Role: Create or select a role for muted members
  • Security Level: Choose between Low, Medium, or High protection

Anti-Raid Protection

The anti-raid system monitors join patterns to detect and prevent coordinated attacks.

Commands

Command Description
/anti-raid enable Enable anti-raid protection
/anti-raid disable Disable anti-raid protection
/anti-raid settings Configure sensitivity and actions
/lockdown Manually lock the server

Anti-Spam System

Automatically detect and handle spam messages including duplicates, mass mentions, and rapid messaging.

Commands

Command Description

/anti-spam settings Configure thresholds and actions

Moderation Commands

Comprehensive moderation toolkit for managing your community.

Command Description
/ban <user> [reason] Ban a user from the server
/kick <user> [reason] Kick a user from the server
/timeout <user> <duration> Timeout a user
/warn <user> <reason> Issue a warning
/purge <amount> Delete multiple messages
/cases <user> View moderation history
