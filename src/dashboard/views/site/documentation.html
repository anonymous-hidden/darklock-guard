Adding the Bot

To add DarkLock to your Discord server:

  • Visit the Add Bot page or use the invite link
  • Select your server from the dropdown
  • Authorize the required permissions
  • Run /setup in your server to configure

Required Permissions

DarkLock requires the following Discord permissions to function properly:

  • Manage Messages ‚Äî Delete spam and rule violations
  • Ban Members ‚Äî Remove malicious users
  • Kick Members ‚Äî Remove problematic users temporarily
  • Manage Roles ‚Äî Assign verification and auto-roles
  • View Audit Log ‚Äî Track server events

Anti-Spam Protection

Automatically detects and removes spam messages, including:

  • Message flooding
  • Duplicate content
  • Mention spam
  • Link spam

Configure with: /anti-spam settings

Anti-Raid System

Protects your server from coordinated attacks with automatic detection and lockdown capabilities.

Enable with: /anti-raid enable

Secure your account with optional 2FA using authenticator apps. Enable this in your account settings for enhanced security.
