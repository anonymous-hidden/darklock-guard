FAQ

Frequently Asked Questions

How do I add DarkLock to my server?

Click the "Add to Discord" button on our website or use the invite link. Select your server from the dropdown, authorize the required permissions, and you're done! Run /setup in your server to configure DarkLock.

What permissions does DarkLock need?

DarkLock requires Administrator permission to function properly. This allows the bot to manage channels, roles, bans, messages, and access audit logs for security features. DarkLock only uses permissions when necessary for its security and moderation features.

Is DarkLock free to use?

Yes! DarkLock offers a free tier with essential security features including anti-spam, anti-raid, basic moderation, and more. Premium plans are available for advanced features like anti-nuke protection, custom branding, and priority support.

How do I configure DarkLock settings?

You can configure DarkLock in two ways: 1) Use slash commands directly in Discord (e.g., /settings, /anti-spam settings), or 2) Use the web dashboard at dashboard for a visual interface with all settings.

Why isn't DarkLock responding to commands?

First, check if DarkLock is online by checking its status in the member list. Make sure DarkLock has permission to read and send messages in the channel. Also ensure you're using slash commands (type "/" to see available commands). If issues persist, try re-inviting the bot with the proper permissions.

How do I set up anti-raid protection?

Use the /anti-raid enable command to enable anti-raid protection. You can then configure the sensitivity and actions with /anti-raid settings. The bot will automatically detect suspicious join patterns and take action based on your settings.

Can I whitelist users from security features?

Yes! You can whitelist trusted users and roles for most security features. For example, /antinuke whitelist @user will exempt a user from anti-nuke protection. Similar options are available for other features in their respective settings.

How do I contact support?

The best way to get support is to join our Discord support server. Our team and community members are active there to help with any questions or issues.
