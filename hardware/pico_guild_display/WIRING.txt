═══════════════════════════════════════════════════════════════════════════
 SERIAL PROTOCOL REFERENCE (115200 baud, newline-delimited)
═══════════════════════════════════════════════════════════════════════════

  Pi → Elegoo:
    LCD:line1|line2       Update LCD (16 chars per line)
    LED1:r,g,b            RGB LED 1 (0-255, threshold 128 → HIGH/LOW)
    LED2:r,g,b            RGB LED 2
    TAMPER:0/1             Tamper LED
    RFID:GREEN/RED/OFF     RFID indicator LEDs
    MATRIX_SCAN/OK/DENIED/ALERT/IDLE/LOCK/BOOT  Dot matrix animations
    PING                   Heartbeat
    CLEAR                  Clear LCD

  Elegoo → Pi:
    READY                  Boot complete
    PONG                   Heartbeat reply
    ACK:cmd                Command acknowledged

  Pi → Pico:
    COUNT:1234             Update guild count
    PING                   Heartbeat
    RESET                  Reset to 0
