üíæ Data Storage & Security

Storage Infrastructure

Your data is stored securely using the following measures:

  • Database: SQLite database with persistent disk storage
  • Password Security: All passwords are hashed using bcrypt with salt
  • Session Management: Secure httpOnly cookies with JWT tokens
  • Backups: Regular automated backups stored securely

Security Measures

  • HTTPS/TLS encryption for all data transmission
  • Rate limiting to prevent abuse and brute-force attacks
  • Comprehensive security headers (CSP, HSTS, XSS protection)
  • Regular security audits and vulnerability assessments
  • Secure file integrity monitoring and tamper detection

Data Persistence: Unlike in-memory databases, your data is persistently stored on disk, ensuring reliability and preventing data loss during service restarts or updates.
