Install Ridgeline on iOS via Safari

This uses the secure web app and installs to Home Screen like a native app.

Open Ridgeline Web App

Open the link in Safari on iPhone or iPad.

  1. Tap the Open Ridgeline Web App button above.
  2. In Safari, tap the Share button.
  3. Choose Add to Home Screen.
  4. Tap Add to install Ridgeline on your home screen.
Tip: if the app is already open in another browser tab, switch to Safari first so Add to Home Screen is available.
