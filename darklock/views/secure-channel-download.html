Darklock Secure Channel

v0.1.0 AVAILABLE NOW
End-to-end encrypted · Zero knowledge · Open protocol

A fully encrypted desktop messaging application built with Tauri + Rust. Every message is encrypted client-side with X25519 key exchange and ChaCha20-Poly1305, with no plaintext ever leaving your device. Designed for teams who need provably secure communication.

X25519 + ChaCha20-Poly1305Modern AEAD encryption, every message
Ed25519 Identity KeysCryptographic identity — no passwords sent
Zero-Knowledge ServerRelay never sees plaintext or keys
Encrypted Key VaultArgon2id KDF, AES-256-GCM local storage
Real-time MessagingWebSocket relay, message persistence
Self-Hosted OptionRun your own IDS + relay servers
