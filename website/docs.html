Darklock Documentation

Everything you need to set up and use Darklock in your Discord server.

Initial Setup

1. Invite the Bot

Use the invite link to add Darklock to your server. You need the Manage Server permission to do this.

2. Run the Setup Command

Once the bot is in your server, run:

/setup

This walks you through configuring:

  • Moderation log channel
  • Alert channel for threats
  • Protection levels
  • Trusted roles

3. Use the Dashboard

For more control, open the web dashboard at admin.darklock.net/dashboard. Log in with Discord, pick your server, and you'll have access to all settings in one place.

Permissions

The bot can only moderate members with roles below its own role in the hierarchy. Place the Darklock role near the top.

User Verification

Darklock includes a verification system with three security profiles. New members are held in an unverified state until they pass the checks.

Standard

Best for casual communities

  • Automatic verification
  • Basic risk checks
  • 7-day account age minimum

High Security

Best for medium-risk servers

  • Captcha challenge required
  • 30-day account age minimum
  • Advanced risk scoring

Ultra Security

Best for high-value servers

  • Manual staff approval
  • Captcha + review notes
  • 90-day account age minimum

Setup

  1. Create an Unverified role (restricted) and a Verified role (full access)
  2. Go to Dashboard > Verification Settings, enable the system, pick a profile, and set the role IDs
  3. Make sure unverified members can only see a welcome/verification channel

Welcome + Verification Flow

  • Welcome ON, Verification OFF — Sends welcome message on join
  • Welcome OFF, Verification ON — Routes user to verification, no welcome
  • Both ON — Welcome is sent after successful verification
  • Both OFF — Normal entry, nothing happens

Anti-Raid

Darklock monitors join rate, account age, username patterns, and avatar status to detect raids. When a raid is detected, the server is locked down automatically.

Default Settings

  • Join rate threshold: 10 users/minute
  • Account age minimum: 7 days
  • Lockdown duration: 30 minutes

All of these can be adjusted in the dashboard.

Manual Controls

/lockdown [duration] [reason] /unlock [reason] /raid-mode [on|off]

Anti-Spam

The spam engine runs seven independent checks on every message: duplicate detection, rate limiting, mention abuse, invite flooding, caps abuse, emoji flooding, and link spam. Messages that trigger checks are removed automatically.

Sensitivity for each check can be tuned in the dashboard under Server > Auto-Moderation.

Anti-Phishing

Darklock maintains a database of known malicious URLs and checks every link posted in your server. Phishing links are removed instantly and the user is warned. The database is updated regularly.

Moderation

Darklock provides a full moderation toolkit with escalation support. Offenses accumulate per user, and the bot can automatically escalate from warning to mute to kick to ban based on your configuration. Offense points decay over time so minor issues don't stack up permanently.

Troubleshooting

Bot isn't responding to commands

  • Make sure the bot has the required permissions
  • Check that the Darklock role is positioned above the roles it needs to manage
  • Confirm commands are being used in a channel the bot can see

Too many false positives on spam detection

  • Lower the sensitivity for the specific check in the dashboard
  • Add trusted roles to the whitelist
  • Adjust the duplicate message threshold

Dashboard not loading

  • Clear your browser cache and cookies
  • Make sure you have the right Discord permissions
  • Try an incognito/private window
