# Darklock Brand Assets

This directory contains the official Darklock logo and brand assets.

## Files

- **darklock-logo.svg** - Primary logo for dark backgrounds
- **darklock-logo-light.svg** - Logo variant for light backgrounds  
- **darklock-icon.svg** - Simplified icon for small sizes (favicons, app icons)

## Usage Guidelines

### Logo Variants

- **Dark Mode (default)**: Use `darklock-logo.svg`
- **Light Mode**: Use `darklock-logo-light.svg`
- **Icons/Favicons**: Use `darklock-icon.svg`

### Color Palette

The logo uses a blue-to-purple gradient reflecting trust, security, and innovation:

- **Sky Blue**: `#60A5FA`
- **Indigo**: `#6366F1`
- **Deep Purple**: `#7C3AED`
- **Background Dark**: `#020617`

### Do Not

- Do not modify the logo shape or proportions
- Do not change the gradient colors
- Do not add effects like shadows or outlines
- Do not use on busy backgrounds without proper contrast

### Zero-Trust & Safe Mode Colors

**Important**: These semantic colors must remain unchanged:

- **Zero-Trust Badge**: `#ec4899` (pink) - indicates Zero-Trust enforcement
- **Safe Mode Badge**: `#f59e0b` (amber) - indicates Safe Mode active
- **Success**: `#10b981` (emerald)
- **Error**: `#ef4444` (rose)

## Sizing Recommendations

- **Website Header**: 32-40px height
- **Hero/Landing**: 120-160px height
- **Footer**: 24-32px height
- **Favicon**: 16x16, 32x32, 64x64px
- **Desktop App**: 128x128, 256x256px (see tauri icons)

## Technical Notes

- All SVG files are optimized for web use
- Gradients use relative percentages for scalability
- Dark background cutouts use `#0a0a0f` (near-black)
- Light background cutouts use `#ffffff` (white)
