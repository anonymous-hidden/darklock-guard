YOUR PRIVACY IS OUR PRIORITY

We believe in complete transparency about how this software operates and what
data is—or more importantly, is NOT—collected.

1. DATA COLLECTION

   IMPORTANT: This application connects to Darklock servers for authentication
   and service functionality.
   
   Data Collected:
   • Login credentials (username, password) for authentication
   • Account information (email, profile data)
   • Service usage and activity logs
   • System information for compatibility and security
   
   Data NOT Collected:
   ✓ NO advertising or behavioral tracking for third parties
   ✓ NO sale of personal data
   ✓ NO unauthorized access to files or system data

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

2. LOCAL STORAGE

   Local data stored on your device:
   
   • %APPDATA%\Darklock\config.json - Installation preferences and settings
   • Authentication tokens and session data
   • Cached application data for performance
   
   Some data is synchronized with our servers to provide cross-device
   functionality and backup services.

3. NETWORK CONNECTIONS
application connects to:
   
   DARKLOCK SERVERS:
   • Authentication and login services (darklock.net)
   • User account management and settings sync
   • Service APIs for application features
   • Security monitoring and threat detection
   
   THIRD-PARTY SERVICES:
   • Microsoft's winget repositories (for package installation)
   • GitHub releases (for downloading official software)
   • Official vendor CDNs (Docker, Python, Node.js, etc.)
   
   All connections to Darklock servers are encrypted using TLS/SSL.

4. THIRD-PARTY SOFTWARE

   The packages installed through this wizard (Git, VS Code, Docker, etc.)
   are governed by their own privacy policies. We recommend reviewing:
   
   • Microsoft Visual Studio Code: https://code.visualstudio.com/privacy
   • Docker Desktop: https://www.docker.com/legal/privacy
   • Git: https://git-scm.com/
   
   DATA SECURITY & RETENTION

   We take security seriously:
   
   • All server communications use industry-standard encryption (TLS 1.3+)
   • Passwords are hashed using bcrypt with strong salts
   • Authentication tokens expire and are regularly rotated
   • Access logs are maintained for security auditing
   
   Data Retention:
   • Account data: Retained while account is active
   • Activity logs: 90 days rolling retention
   • Deleted accounts: Data purged within 30 days
   • Code transparency
   • No hidden processes
   Under applicable privacy laws (GDPR, CCPA), you have the right to:
   
   ✓ Access your personal data stored on our servers
   ✓ Request correction of inaccurate data
   ✓ Request deletion of your account and associated data
   ✓ Export your data in a portable format
   ✓ Opt-out of non-essential data collection
   ✓ Uninstall the software at any time
   
   DATA SHARING

   We do NOT sell your personal data. Data may be shared only in these cases:
   
   • With your explicit consent
   • To comply with legal obligations (court orders, subpoenas)
   • To protect our rights, property, or safety
   • With service providers bound by confidentiality agreements
